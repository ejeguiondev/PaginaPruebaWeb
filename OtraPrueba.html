HTMl es una herramienta bastante util para poder hacer la estructura de tu proyecto, ademas

Css

Css es una herramienta para darle estilo a tus Elementos, este es muy
usado con HTML y este sirve para dar tanto color, como forma tamaño...
etc. Una herramienta inpresindible si es usada muy bien.
